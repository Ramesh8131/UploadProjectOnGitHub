# Upload Project on gitHub use following steps::::

1. Create Repository On github

2. open cmd terminals.

3. Go to Current working directory

4) git init

5) git add .

6) git commit -m "Add all my files"

7) git remote add origin https://github.com/yourusername/your-repo-name.git

8) git push -u origin master

also show list of recent commits
